# Employee Attrition Prediction System (Task-1)

## Objective
Predict whether an employee is likely to leave a company using machine learning.

## Key Features Implemented
- Dataset Selection: Public HR Analytics dataset
- Feature Engineering & preprocessing
- Classification Models:
    - Logistic Regression
    - Decision Tree
    - Random Forest
- Evaluation using Accuracy, Precision, Recall
- Visualization using graphs

## Python Version
Tested on Python 3.10

## Run Instructions
pip install -r requirements.txt
python main.py
